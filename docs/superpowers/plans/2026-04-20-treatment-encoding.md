# Treatment Class Encoding Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Step 10 to `FeatureTransformer` that encodes `treatment_class` string → integer via a versioned `treatment_encoding.json` artifact, making the Parquet feature matrix fully numeric and model-ready for tree-based classifiers.

**Architecture:** A module-level `build_treatment_encoding(treatment_dict)` function derives a deterministic `str→int` map by sorting `treatment_dict.json` keys alphabetically. `FeatureTransformer.__init__` loads `treatment_encoding.json` as a 4th artifact and applies it in new Step 10 after the existing column allowlist check. `feature_engineering.py` generates and writes the artifact before calling `transform()`.

**Tech Stack:** Python 3, pandas, json, pytest, DVC

**Critic agent:** `.claude/agents/denttime-critic.md` — run after each task.

```python
def build_treatment_encoding(treatment_dict: dict) -> dict:
    """Deterministic str→int encoding: sorted dict keys → 0…N-1."""
    return {cls: i for i, cls in enumerate(sorted(treatment_dict.keys()))}
```
